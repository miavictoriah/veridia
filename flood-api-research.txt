Flood Risk API Research:

1. Environment Agency Flood Monitoring API (environment.data.gov.uk) - currently 503 error, but provides real-time flood warnings and alerts
2. Gov.uk Long Term Flood Risk service (check-long-term-flood-risk.service.gov.uk) - postcode-based lookup
3. Flood Map for Planning (flood-map-for-planning.service.gov.uk) - planning-focused flood zones
4. PropertyData API (propertydata.co.uk) - commercial, paid
5. PropEco Flood Risk API - commercial
6. GitHub flood-risk project (djtarrant/flood-risk) - open source postcode lookup
